# Contributing

Keep this portfolio honest, accessible, and lightweight.

1. Use Node from `.nvmrc` and run `npm ci`.
2. Keep project content in `src/data/projects.js`; do not duplicate cards in HTML.
3. Preserve the 17-card inventory and private-client safeguards.
4. Run `npm run check` and `npm run test:e2e` before requesting review.
5. Include responsive evidence for visual changes.

Do not add unverified metrics, private paths, secrets, a contact form endpoint, or source-related UI for the private client project.
