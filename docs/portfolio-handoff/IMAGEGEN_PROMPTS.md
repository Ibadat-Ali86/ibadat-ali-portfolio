# IMAGEGEN_PROMPTS.md

## Global art direction

Generate one coherent family of 16:10 WebP images. Use warm paper, near-black, acid lime, cobalt, technical diagrams, screen-print texture, and hard-edged modular geometry. Do not place project titles or UI copy inside the artwork. Reject fake logos, gibberish text, photorealistic identifiable people, medical gore, watermarks, and generic 3D gradients.

Create `public/assets/generated/PROVENANCE.md` with final filename, prompt heading, generation date, and any manual crop/compression work.

## Shared non-project assets

Abstract end-to-end AI system blueprint: data streams, model nodes, API boundaries, deployment blocks, monitoring signals, and feedback loops; controlled brutalist composition; no text.

### about-topology-portrait.webp
Abstract human-and-system collaboration portrait without depicting a real person: geometric silhouette, topology contours, graph nodes, code-like blocks, warm paper and cobalt; no face details and no text.

### noise-tile.webp
Seamless subtle monochrome screen-print grain, low contrast, suitable for repeating overlay.
