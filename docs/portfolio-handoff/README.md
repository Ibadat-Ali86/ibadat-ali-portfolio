# Ibadat Ali — Brutalist AI Systems Portfolio

This is the **final Codex implementation handoff** for the complete portfolio. It resolves repository duplication, private-client visibility, project hierarchy, copy, design rules, architecture, media generation, and launch verification.

## Final decisions

- Keep valuable child projects as independent case studies even when their parent collection repositories are hidden.
- Hide `data-science-portfolio` as a collection, while retaining VITAL-LINK and Pakistan E-commerce Price Prediction.
- Use **AdaptIQ / ForecastAI** as the single canonical Walmart forecasting case study; describe the standalone Walmart repository only as an earlier development stage.
- Feature **Evershine Academy LMS** as a private client project with a live-site CTA and **no GitHub/source link anywhere**.
- Exclude forks, uncertain-ownership repositories, empty repositories, duplicate shells, and collection-only repositories.
